Create (GET)

CRUD App - Create (GET)

Add a form to the web app that will allow the user to insert a new player into the database. Create an "Add Player" page that will be accessible via a GET request.

Creating the GET Route Handler

To properly handle the GET request for the "Add Player" page, it is necessary to build a handler function that renders the EJS.

  1. In the "views" folder, create a new file named "edit-player.ejs"
    • For testing purposes, add an HTML header to the file saying "Add Player"
  2. In the routes folder, create a new file named player.js
  3. In the player.js file, set module.exports equal to a new empty object
  4. In the module.exports object, set a property named addPlayerPage to a new function with request and response parameters
  5. In the body of the addPlayerPage function, call response.render and pass in "edit-player"

player.js

module.exports = {
    // Load the form to add a player - GET
    addPlayerPage: function (request, response) {
        // Load the page
        response.render('edit-player');
    }
}

Hooking up the Route

Update the app so that /add loads up the "Add Player" page, and there is a link to /add in the nav bar.

  1. In the app.js file, under the index module, require the ./routes/player module and put it in const player
  2. Under the app.get call, add another app.get to hook up /add to player.addPlayerPage
  3. In the header.ejs file, add an a to the navbar that links to the /add page
     <a class="float-right" href="/add">Add a Player</a>
  4. Load up the homepage, click the "Add a Player" link, and verify that the route loads the HTML from "edit-player.ejs"

Filling out the "Add Player" EJS

Update the "edit-player.ejs" file so that it properly renders an HTML form that takes in Soccer Player information.

  1. Replace the test HTML in "edit-player.ejs" with EJS to include the header.ejs partial
  2. Create a div with a class of "container"
  3. Within the "container" div, create a form that will POST to add/ on submit
    • Set its method to "post"
    • Set its action to "add/"
  4. In the form, add a div element with a class of "form-row"
  5. In the "form-row" div, create a "form-group" div holding the label and input for First Name:
     <div class="form-group col-md-6">
     <label for="first-name">First Name</label>
     <input type="text" class="form-control" required>
 </div>
  6. Add a name attribute to the input element with a value of "first_name"
    • This is how the form knows what data to send the server in the POST request
  7. Under the First Name "form-group" div, create another "form-group" div for Last Name
  8. Under the "form-row" div, create another "form-row" div
  9. In the second "form-row" div, create a "form-group" for Number
    • Ensure that the type of the input element is "number"
  10. Create another "form-group" for Position using a select to provide the user with a dropdown list, including:
    • Goalkeeper
    • Defender
    • Midfielder
    • Forward
  11. Under the second "form-row" div, add HTML for a submit button:
     <button type="submit" class="btn btn-success float-right">Add Player</button>
  12. Load up the "Add Player" page, and verify that the form appears and submits the proper POST request!
